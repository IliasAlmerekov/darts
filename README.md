# Dart score app
Dart score app is an app where you can effortlesly keep track of your dart games.

## Installation

1. Clone the repository:
```bash
 git clone ""
```

2. Install dependencies:
```bash
 npm install
 ```

## Getting Started
```bash
npm start
```
